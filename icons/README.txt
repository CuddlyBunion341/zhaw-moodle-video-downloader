Icon files (icon-48.png and icon-96.png) should be placed in this directory.

You can create icons using any image editor. Recommended sizes:
- icon-48.png: 48x48 pixels
- icon-96.png: 96x96 pixels

Suggested design: A simple, clean icon with a clipboard or video symbol.
Keep it minimal and professional.

Online tools to create icons:
- https://favicon.io/
- https://www.canva.com/
- Or use any PNG image resized to the appropriate dimensions
